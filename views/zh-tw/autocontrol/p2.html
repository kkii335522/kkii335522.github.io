PIR感應自動樓梯燈(雙切)開關

產品特性
1. 人來燈就亮，不用摸黑找開關，踩空樓梯
2. 有延遲關燈時間，人離開燈就滅。
3. 不必碰觸開關，防觸電，不會摸髒牆壁。
4. 與原配線相同，免重新拉線。
5. 外觀與傳統開關相同，可直接替換。
6. 安裝簡單，可自行DIY。

產品簡介
1. 您是否曾經在夜晚要上樓時，摸黑找開關不小心撞到階梯，要下樓時摸黑找開關，差點踩空樓梯而摔倒。為此本公司針對這問題研發出此一產品，當人一接近樓梯口時，由熱紅外線感應而開啟電燈，保障您夜晚上下樓梯的安全。
2. 本產品是將雙切開關(二個開關控制一個電燈)迴路，改為由熱紅外線自動感應，如樓梯燈或長廊走道燈。
3. 雙切開關迴路就是樓上樓下各有一個開關共同控制一個電燈，而開關一般都會裝在要上下樓梯位置的牆壁上，因此只要直接將雙切開關更換成本產品，不論在上樓或下樓皆可馬上感應開燈。
4. 本產品尺寸大小與傳統開關一樣，配線也與原配線相同，因此不用重新拉線，替換簡單可自行DIY。
5. 全自動感應開燈，延時關燈，不用動手，防止觸電，不會摸髒開關周圍的牆壁。

產品照片

操作說明
1. 內有明暗設計，白天不感應，感應3分鐘後自動熄滅。
2. 按鍵按一下，自動變手動，LED高亮，可手控燈亮或滅。
3. 按鍵長壓3秒，手動變自動，LED微亮。
4. 通電後須熱機10分鐘後才能穩定作用。

產品規格
體積大小 44×22×37 mm
單蕊線 ψ1.6或ψ2.0 mm切掉外皮包覆 18mm
適用電壓 AC 110V
負載範圍 5W～80W傳統或省電燈泡，10W～40W日光燈
感應距離 約3公尺

適用燈泡
1. 日光燈啟動速度慢，較不建議與本產品搭配使用，但是你也可以更換為電子啟動器或電子安定器，並請於燈泡兩端並接上一個0.1uF耐壓400V的電容。
2. 少部份廠牌省電燈泡，因其內部電路設計不同，與本產品搭配會有閃爍現象，請於燈泡兩端並接上一個0.1uF耐壓400V的電容。
3. 傳統鎢絲燈泡任何廠牌皆可使用。

安裝方法
1. 切斷電源，打開樓上、樓下壁上開關蓋板，取出開關，拆除電線，銅線如有氧化生銹，請清潔乾淨。
2. 將樓上電燈出線A插上主控器插孔1，將樓下地線D插上副控器插孔1。
3. 將樓下剩下兩條線的其中一條插上副控器插孔4（B與C兩條電線顏色，大部份皆同色）。
4. 打開電源，將樓上剩下兩條線的其中一條接觸主控器插孔1的電線A，電燈會亮的那一條線為電線C。
5. 切斷電源，將剛測出的電線C插上主控器插孔4。
6. 剩下的電線B插在主控器插孔3與副控器插孔3。
7. 裝好自動開關，蓋上蓋板，打開電源即可測試使用。

注意事項
1. 安裝請配帶絕緣手套，以免發生觸電。
2. 安裝前請詳讀說明書，並照步驟施工，以免損壞本產品。
3. 安裝前請留意本產品之電壓及負載規格。
4. 避免陽光直接曝曬及潮溼。
